# rubikvn

## Deploy with local Node
1. Install nodejs on your machine
2. Run 
```
npm install
npm start
```
3. Open [http://localhost:3000](http://localhost:3000)

## Deploy with Docker
1. Install Docker and docker-compose
2. Run
```shell
docker-compose up
```
3. Open [http://localhost:3000](http://localhost:3000)
